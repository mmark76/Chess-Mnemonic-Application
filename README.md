# Chess Mnemonic Application

A static browser-based application for studying chess games through mnemonic structures, memory palaces, associations, shortnames, PAO systems, SAN text, and story-based recall.

The application transforms a chess game into structured memory material: moves, squares, pieces, actions, loci, anchors, associations, shortnames, PAO rows, and narrative output.
