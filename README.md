# 🇬🇧 **README — Chess Mnemonic Application & Epic Chess Stories Creator**

A modular **web-based mnemonic system** that transforms chess moves into structured **Mnemonic Histories** using Temporal, Spatial, Character-based, PAO, Verse, and Foundational memory layers.
The application helps users encode games, study patterns, and build personalized mnemonic structures.

---

## **1. Overview**

This project provides a unified interface for:

* Converting PGN or SAN chess moves into mnemonic descriptions
* Viewing, editing, and customizing mnemonic histories
* Using built-in memory libraries (Temporal, Spatial, Characters, PAO, Verses, Foundations)
* Managing user-defined mnemonic overrides
* Training individual systems through a separate **Flashcards Trainer**

All logic runs locally in the browser — no database, no server.

## **3. Core Functionality**

### **3.1 Input Handling**

* Load complete PGN games
* OR type moves manually in SAN notation
* Automatic parsing through `pgn.js`

### **3.2 Mnemonic Generation**

Each move maps to:

* **Temporal mnemonic** (sequence logic)
* **Spatial mnemonic** (board structure / loci)
* **Character mapping** (piece personalities)
* **PAO encoding**
* **Verse/poetic associations**
* **Foundational patterns**

Generated by the libraries inside `/js/libraries/`.

### **3.3 Mnemonic History Panel**

* Shows move-by-move stories
* Allows **manual overrides** saved locally
* Updates dynamically when loading new games

### **3.4 User Custom Library**

`user_custom.js` allows users to override the built-in libraries with completely personalized mappings.

## **7. Philosophy**

The system combines:

* **Spatial memory** (locations → positions)
* **Temporal memory** (sequences → moves)
* **Narrative memory** (characters → pieces)
* **Associative memory** (PAO → encoding)

Each chess move becomes a cognitive “anchor” that fits into a larger internal architecture.

# 🇬🇷 **README — Εφαρμογή Μνημονικών Σκακιού & Δημιουργός Επικών Ιστοριών — v3.3**

Μια modular **διαδικτυακή εφαρμογή μνήμης**, που μετατρέπει τις κινήσεις σκακιού σε δομημένες **Μνημονικές Ιστορίες** μέσα από χρονικά, χωρικά, προσωποποιημένα, PAO, έμμετρα και θεμελιώδη μοτίβα μνήμης.

---

## **1. Επισκόπηση**

Η εφαρμογή επιτρέπει:

* Φόρτωση PGN ή εισαγωγή κινήσεων SAN
* Αυτόματη παραγωγή μνημονικών περιγραφών
* Προβολή και επεξεργασία ιστορικού
* Χρήση έτοιμων βιβλιοθηκών (Temporal, Spatial, Characters, PAO, Verses, Foundations)
* Προσωποποιημένα overrides σε κάθε κίνηση
* Μελέτη μνημονικών συστημάτων μέσω Flashcards

Όλα λειτουργούν **τοπικά** στο πρόγραμμα περιήγησης.

## **3. Πυρήνας Λειτουργιών**

### **3.1 Εισαγωγή Δεδομένων**

* Φόρτωση PGN
* Εισαγωγή χειροκίνητων SAN κινήσεων

* PAO κωδικοποίηση
* Έμμετρο μοτίβο
* Θεμελιώδη συνειρμικά στοιχεία

* Προβολή κειμένου ανά κίνηση
* Δυνατότητα χειροκίνητης αντικατάστασης
